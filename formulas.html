דף נוסחאות

1 · יסודות הסתברות

בסיס לכל הקורס. השתמש לכל שאלת הסת׳ ראשונית: מותנית כשנתון "אם X קרה", הכפלה כש"א וגם ב", שלמה+בייס כשיש חלוקה לתתי־מקרים.

אקסיומות: $0 \le P(A) \le 1$, $P(\Omega)=1$

זרים: $P(A \cup B)=P(A)+P(B)$.

כללי: $P(A \cup B)=P(A)+P(B)-P(A \cap B)$, $P(A^c)=1-P(A)$.

מותנה והכפלה $$P(A \mid B) = \tfrac{P(A \cap B)}{P(B)}$$ $$P(A \cap B) = P(A) P(B \mid A)$$ הסת׳ שלמה + בייס $$P(A) = \sum_i P(A \mid B_i) P(B_i)$$ $$P(B_j \mid A) = \tfrac{P(A \mid B_j) P(B_j)}{\sum_i P(A \mid B_i) P(B_i)}$$

אי־תלות: $P(A \cap B) = P(A) P(B)$ ⇔ $P(A|B) = P(A)$. זרות ≠ אי־תלות

1 · התפלגויות בדידות

חוסר זיכרון יש רק ל־G (בדידה) ול־exp (רציפה).

B(n,p) — מספר הצלחות מתוך n $$P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}$$ $$E=np, \quad V=np(1-p)$$ G(p) — ניסיון ראשון להצלחה $$P(X=k) = (1-p)^{k-1} p, \quad E=\tfrac{1}{p}, \quad V=\tfrac{1-p}{p^2}$$ U{m,…,n} — אחידה בדידה $$E=\tfrac{m+n}{2}, \quad V=\tfrac{(n-m+1)^2-1}{12}$$ Pois(λ) $$P(X=k) = \tfrac{e^{-\lambda}\lambda^k}{k!}, \quad E=V=\lambda$$

1 · התפלגויות רציפות

U(a,b) $$f(x) = \tfrac{1}{b-a},\ a \le x \le b$$ $$E = \tfrac{a+b}{2}, \quad V = \tfrac{(b-a)^2}{12}$$ exp(λ) $$f(x) = \lambda e^{-\lambda x},\ x \ge 0$$ $$E = \tfrac{1}{\lambda}, \quad V = \tfrac{1}{\lambda^2}$$

חוסר זיכרון: $P(X > s+t \mid X > s) = P(X > t)$

2 · נורמלית + תקנון

השתמש כאשר: ידוע ש־$X \sim N(\mu, \sigma^2)$, או כשרוצים לקרב סכומי בינום/בדיד גדול. שים לב: $\sigma^2$ זו שונות, לא ס"ת — בתקנון מחלקים ב־$\sigma$.

תקנון $$Z = \tfrac{X-\mu}{\sigma} \sim N(0,1)$$ חישוב הסת׳ עם Φ $$P(Z > z) = 1 - \Phi(z), \quad \Phi(-z) = 1 - \Phi(z)$$ $$P(z_1 < Z < z_2) = \Phi(z_2) - \Phi(z_1)$$ אחוזונים $$X_p = \mu + \sigma \cdot z_p, \quad z_p = -z_{1-p}$$

2 · סכום/ממוצע · מ.ה.מ.

מ.ה.מ. — שימוש בקירוב נורמלי: ממוצע n גדול (n≥30) של תצפיות ב"ת מתפלג בקירוב נורמלי מכל התפלגות מקורית.

סכום נורמליים ב"ת $$\sum X_i \sim N\!\left(\sum \mu_i, \sum \sigma_i^2\right)$$ $$\bar X_n \sim N\!\left(\mu, \tfrac{\sigma^2}{n}\right)$$ מ.ה.מ. (n גדול) $$\bar X_n \approx N\!\left(\mu, \tfrac{\sigma^2}{n}\right)$$ קירוב לבינום + תיקון רציפות

תקף כש־$np \ge 5$ ו־$n(1-p) \ge 5$:

$$B(n,p) \approx N(np, np(1-p))$$
  • $P(X=k) \to P(k-0.5 < Y \le k+0.5)$
  • $P(X \le k) \to P(Y \le k+0.5)$
  • $P(X \ge k) \to P(Y \ge k-0.5)$

3 · שרשרת מרקוב

תכונת מרקוב: העתיד תלוי רק בהווה. השתמש בשיווי משקל רק אם השרשרת רגולרית (קיים m כך ש־$A^m$ חיובית).

מטריצת מעבר A: $A_{i,j}=P(X_{n+1}=j \mid X_n=i)$. שורות מסתכמות ל־1.

m צעדים $$\varphi_m = \varphi_0 \cdot A^m$$ שיווי משקל (רגולרית) $$\pi A = \pi, \quad \sum_i \pi_i = 1$$

תנאי רגולרית ⇔ אי־פריקה + אפריודית. רק אז קיים $\pi$ יחיד.

4 · HMM

איזה אלגוריתם? מסלול חבוי הסביר ביותר ← Viterbi. הסת׳ סדרת תצפיות ← Forward. הסת׳ מצב בזמן i ספציפי ← Posterior.

רכיבים: A (מעבר), B (פלט תלוי במצב), S (התחלה).

$$P(X,H) = S_{h_1} B_{h_1,x_1} \prod_{i=1}^{n-1} A_{h_i,h_{i+1}} B_{h_{i+1},x_{i+1}}$$ Viterbi (max) $$V_1(\ell) = S_\ell B_{\ell, x_1}$$ $$V_{i+1}(\ell) = B_{\ell, x_{i+1}} \max_k \{V_i(k) A_{k,\ell}\}$$ Forward (∑) $$f_{i+1}(\ell) = B_{\ell, x_{i+1}} \sum_k f_i(k) A_{k,\ell}$$ $$P(X) = \sum_\ell f_n(\ell)$$ Posterior $$P(H_i=k \mid X) = \tfrac{f_i(k) b_i(k)}{P(X)}$$

log Viterbi: עובדים ב־$\log$ למניעת underflow — סוכמים במקום להכפיל.

5 · סטטיסטיקה תיאורית

סולמות מדידה: שמי (נומינלי) ← רק שכיח. סדר (אורדינלי) ← גם חציון/אחוזונים. כמותי ← גם ממוצע ושונות.

מדדי מרכז $$\bar X = \tfrac{\sum X_i}{n}, \quad Me = X_{0.5}, \quad Q_1, Q_3$$ פיזור (תיאורית, ÷ n) $$\sigma_X^2 = \tfrac{\sum(X_i-\bar X)^2}{n} = \tfrac{\sum X_i^2}{n} - \bar X^2$$ $$\text{IQR} = Q_3 - Q_1$$ התמרה לינארית Y = aX + b $$\bar Y = a\bar X + b, \quad \sigma_Y^2 = a^2 \sigma_X^2$$

זיהוי הטיה:

  • זנב ימני: $Mo < Me < \bar X$
  • סימטרית: כולם שווים
  • זנב שמאלי: $\bar X < Me < Mo$

6 · יסודות בדיקת השערות

מסגרת לפרקים 7–10. H₀ = מצב קיים. נטל ההוכחה על מי שמכחיש. "לא דחינו" ≠ "H₀ נכונה".

  • טעות I: דחיית H₀ נכון. $\alpha = P_{H_0}(C)$
  • טעות II: אי־דחייה כש־H₁. $\beta = P_{H_1}(\bar C)$
  • עוצמה: $\pi = 1 - \beta = P_{H_1}(C)$

טרייד־אוף אם α גדלה, β קטנה — ולהפך. הנוהל: קובעים α מראש (0.05 / 0.01 / 0.1) ובוחרים את המבחן עם β המינימלי.

למת ניימן־פירסון — מבחן חזק ביותר $$\lambda(x) = \tfrac{L_{H_1}(x)}{L_{H_0}(x)}, \ C^* = \{\lambda > K\}$$ החלטה דרך p-value $$p.v \le \alpha \Leftrightarrow \text{דחה}$$

7 · מבחן z על תוחלת

השתמש כאשר: $\sigma$ באוכלוסייה ידועה, ויש נורמליות (או n גדול לפי מ.ה.מ.). אם $\sigma$ לא ידועה ← פרק 8.

$$Z = \tfrac{\bar X_n - \mu_0}{\sigma/\sqrt n} \sim_{H_0} N(0,1)$$ חד־צדדית שמאלית $$C = \{\bar X_n \le \mu_0 - Z_{1-\alpha} \sigma/\sqrt n\}$$ חד־צדדית ימנית $$C = \{\bar X_n \ge \mu_0 + Z_{1-\alpha} \sigma/\sqrt n\}$$ דו־צדדית $$C = \{|\bar X_n - \mu_0| \ge Z_{1-\alpha/2} \sigma/\sqrt n\}$$ רווח סמך 1−α $$\left[\bar X_n \pm Z_{1-\alpha/2} \tfrac{\sigma}{\sqrt n}\right]$$

קשר CI ↔ בדיקה דו־צדדית: $\mu_0$ ברווח ← לא דחה. $\mu_0$ מחוץ ← דחה.

אורך הרווח: $L = 2 Z_{1-\alpha/2} \sigma/\sqrt n$. לחתוך לחצי — n פי 4.

8 · t — מדגם בודד

השתמש כאשר: $\sigma$ לא ידועה, ויש נורמליות. דרגות חופש = n−1.

$$S^2 = \tfrac{\sum(X_i - \bar X)^2}{n-1} = \tfrac{\sum X_i^2 - n\bar X^2}{n-1}$$ $$T = \tfrac{\bar X_n - \mu_0}{S/\sqrt n} \sim_{H_0} t_{n-1}$$

אזורי דחייה זהים ל־z אבל עם $t_{n-1, 1-\alpha}$ במקום $Z_{1-\alpha}$.

רווח סמך 1−α $$\left[\bar X_n \pm t_{n-1, 1-\alpha/2} \tfrac{S}{\sqrt n}\right]$$

8 · t מזווג + 2 ב"ת + F

זווג vs ב"ת: כל נבדק תורם 2 ערכים (לפני/אחרי) ← מזווג. שתי קבוצות שונות ← ב"ת.

מזווג (D = X − Y) $$T = \tfrac{\bar D - \mu_{D,0}}{S_D/\sqrt n} \sim t_{n-1}$$ F לשונויות (מקדים ב"ת) $$F = \tfrac{S_X^2}{S_Y^2} \sim F(n_X-1, n_Y-1)$$ pooled (לא דחינו F - שונויות שוות) $$S_p^2 = \tfrac{(n_X-1) S_X^2 + (n_Y-1) S_Y^2}{n_X + n_Y - 2}$$ $$T = \tfrac{\bar Y - \bar X}{\sqrt{S_p^2(1/n_X + 1/n_Y)}} \sim t_{n_X+n_Y-2}$$

דחינו F ← Welch (Phyton).

9 · א־פרמטריים

השתמש כאשר: סולם סדר, התפלגות לא נורמלית, או מדגם קטן עם חשד. תמיד חלשים יותר מהפרמטרי אם הוא תקף.

Wilcoxon מזווג — חליף ל־t מזווג

n* = #זוגות עם הפרש ≠ 0. דרגים על $|D_i|$. R₊ = סכום דרגות חיוביות.

$$R_+ + R_- = \tfrac{n^*(n^*+1)}{2}$$

תחת H₀ (n* ≥ 12):

$$R_+ \sim N\!\left(\tfrac{n^*(n^*+1)}{4}, \tfrac{n^*(n^*+1)(2n^*+1)}{24}\right)$$ מבחן הסימן — סולם סדר בלבד $$S_+ \sim_{H_0} B(n^*, 0.5) \approx N(0.5 n^*, 0.25 n^*)$$ Mann−Whitney — 2 ב"ת

מדרגים את 2 הקבוצות יחד. $R_X$ = סכום דרגות מ־X.

$$R_X + R_Y = \tfrac{n(n+1)}{2}$$

תחת H₀ (max ≥ 10):

$$R_X \sim N\!\left(\tfrac{n_X(n+1)}{2}, \tfrac{n_X n_Y (n+1)}{12}\right)$$ $$M_X = R_X - \tfrac{n_X(n_X+1)}{2}$$

10 · χ² — טיב התאמה ואי־תלות

שני שימושים: טיב התאמה (האם התפלגות תיאורטית מתאימה למדגם?) ואי־תלות (האם 2 משתנים שמיים תלויים?). תמיד חד־צדדי ימני ← ערך גבוה = דחייה.

$$\chi^2_s = \sum \tfrac{(O_i - E_i)^2}{E_i} \sim \chi^2_{df}$$ טיב התאמה

$E_i = p_i \cdot n$, df = k−1.

אי־תלות (טבלת r × c) $$E_{i,j} = \tfrac{(\text{שורה } i)(\text{עמודה } j)}{n}$$

df = (r−1)(c−1).

דרישה בכל תא $E \ge 5$. אחרת — איחוד קטגוריות.

🧭 בחירת מבחן — פלואו מהיר

  • תוחלת, σ ידועה, נורמלי ← z (פרק 7)
  • תוחלת, σ לא ידועה, נורמלי ← t חד־מדגם (פרק 8)
  • תוחלת, לא נורמלי, סולם סדר ← Sign על Me (פרק 9)
  • זוגות מזווגים, נורמלי ← t מזווג (פרק 8)
  • זוגות מזווגים, סימטרי, לא נורמלי ← Wilcoxon (פרק 9)
  • זוגות מזווגים, סולם סדר ← Sign (פרק 9)
  • 2 קבוצות ב"ת, נורמלי ← ראשית F, אז pooled t או Welch t (פרק 8)
  • 2 קבוצות ב"ת, לא נורמלי / סדר ← Mann−Whitney (פרק 9)
  • התפלגות תיאורטית? ← χ² טיב התאמה (פרק 10)
  • תלות בין 2 משתנים שמיים? ← χ² אי־תלות (פרק 10)

בכל מבחן: ניסוח H₀ ו־H₁ ← חישוב סטטיסטי ← קביעת p.v. ← השוואה ל־α.
