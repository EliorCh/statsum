מבחני t
(σ לא ידועה) · מבחן F לשונויות

המבחנים השכיחים בפועל: מדגם בודד, מזווג, ושני מדגמים בלתי־תלויים.

בפרק 7 הנחנו ש־σ ידועה — מה שכמעט אף פעם לא קורה במציאות. כאן ה־σ לא ידועה, ונאמד אותה באמצעות S מהמדגם. השינוי המרכזי: התפלגות הסטטיסטי כבר לא נורמלית סטנדרטית, אלא התפלגות t עם n − 1 דרגות חופש. ככל ש־n גדל, t מתקרבת ל־N(0,1). הפרק מציג שלושה מצבים: מדגם בודד, מדגם מזווג (לפני/אחרי על אותם נבדקים), ושני מדגמים בלתי־תלויים. למקרה האחרון יש מבחן מקדים — F לשונויות — שמכריע אם להשתמש ב־pooled t או ב־Welch t.

1 · התפלגות t

📘הגדרה ותכונות

אם X1, …, Xn מתפלגים N(μ, σ2) בלתי תלויים, ו־S2 הוא האומד החסר־הטיה לשונות:

$$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar X)^2$$

אז:

$$T = \frac{\bar X - \mu}{S/\sqrt n} \sim t_{n-1}$$

תכונות tk:

  • סימטרית מסביב ל־0.
  • שונות > 1 (יותר פיזור מ־Z, "זנבות עבים").
  • ככל ש־k (=דרגות חופש) גדל ← tk ← N(0,1). בערך מ־k≥30 כבר אפשר לקרב.
  • אין נוסחה סגורה — משתמשים בטבלאות או scipy.
⚠️חלוקה ב־n − 1, לא ב־n

בפרק 5 (סטטיסטיקה תיאורית) חישבנו שונות עם חלוקה ב־n. כאן (הסקה) חלוקה ב־n−1. ההסבר האינטואיטיבי: כשמשתמשים ב־X̄ של המדגם במקום ב־μ האמיתי, "לקחנו" דרגת חופש אחת. כדי לקבל אומד חסר־הטיה לשונות האוכלוסייה — חייב n−1.

התפלגות t סימטרית מסביב ל־0, אבל עם זנבות עבים יותר מ־N(0,1). ככל ש־df גדלים — tdf מתקרבת לנורמלית. מעל df ≈ 30 ההבדל כמעט לא ניכר.

2 · מבחן t למדגם בודד

📐סטטיסטי, אזורי דחייה, p-value

בדיקת השערה על תוחלת μ כאשר σ לא ידועה.

סטטיסטי המבחן:

$$T = \frac{\bar X_n - \mu_0}{S/\sqrt n} \sim t_{n-1}$$

אזורי דחייה (אותו רעיון כמו ב־z, רק עם tn−1,1−α):

חד־צדדי שמאלי (H1: μ < μ0):

$$C = \{ T \le -t_{n-1,1-\alpha} \}$$

חד־צדדי ימני (H1: μ > μ0):

$$C = \{ T \ge t_{n-1,1-\alpha} \}$$

דו־צדדי (H1: μ ≠ μ0):

$$C = \{ |T| \ge t_{n-1,1-\alpha/2} \}$$

p-value מחושב על אותו רעיון:

$$p.v(x_s) = \begin{cases} P(t_{n-1} \le t_s) & \text{שמאלי} \\ P(t_{n-1} \ge t_s) & \text{ימני} \\ 2\min\{P(t_{n-1} \le t_s),\, P(t_{n-1} \ge t_s)\} & \text{דו-צדדי} \end{cases}$$

📐רווח סמך ל־μ ברמת סמך 1 − α

$$\left[\ \bar X_n - t_{n-1,1-\alpha/2} \cdot \frac{S}{\sqrt n},\ \ \bar X_n + t_{n-1,1-\alpha/2} \cdot \frac{S}{\sqrt n}\ \right]$$

שווה לחלוטין לרווח עם Z, רק σ מוחלף ב־S, ו־Z מוחלף ב־tn−1.

3 · מדגם מזווג (paired t-test)

📘מתי משתמשים

כל נבדק תורם זוג תצפיות: לפני/אחרי, ימין/שמאל, טיפול/פלצבו על אותו אדם. הרעיון: מחשבים את ההפרש בכל זוג, ומריצים מבחן t רגיל על ההפרשים.

נסמן Di = Xi − Yi. תחת H0: μX = μY ⇔ μD = 0. המבחן הוא בעצם מבחן t על D עם μ0 = 0.

📐נוסחאות

$$\bar D = \frac{1}{n}\sum D_i, \quad S_D^2 = \frac{1}{n-1}\sum (D_i - \bar D)^2$$

$$T = \frac{\bar D - 0}{S_D/\sqrt n} \sim t_{n-1}$$

רווח סמך להפרש התוחלות:

$$\left[ \bar D - t_{n-1,1-\alpha/2} \cdot \frac{S_D}{\sqrt n},\ \bar D + t_{n-1,1-\alpha/2} \cdot \frac{S_D}{\sqrt n} \right]$$

⚠️מזווג ולא בלתי־תלוי

אם המדדים נלקחו מאותם נבדקים (לפני וגם אחרי) — מזווג. הפרשים מבטלים שונות בין־אישית והופכים את המבחן להרבה יותר עוצמתי.

אם זה שתי קבוצות שונות של נבדקים (קבוצת ביקורת + קבוצת ניסוי) — בלתי־תלויים (סעיף 4).

4 · שני מדגמים בלתי־תלויים

📘שתי גרסאות

שתי קבוצות נפרדות, מדגמים X1, …, XnX ו־Y1, …, YnY נורמליים בלתי־תלויים. H0: μX = μY.

שתי האפשרויות:

  1. pooled t (שונויות שוות): מניחים σX2 = σY2, מאמדים את השונות המשותפת ממוצעת ממושקלת.
  2. Welch t (שונויות שונות): לא מניחים שוויון, מתאמים את דרגות החופש.

איך מחליטים? מבחן F מקדים (סעיף 5).

📐pooled t (שוויון שונויות)

אומד משותף לשונות:

$$S_p^2 = \frac{(n_X - 1)S_X^2 + (n_Y - 1)S_Y^2}{n_X + n_Y - 2}$$

סטטיסטי:

$$T = \frac{(\bar Y - \bar X) - (\mu_Y - \mu_X)}{\sqrt{S_p^2 \left(\frac{1}{n_X} + \frac{1}{n_Y}\right)}} \sim t_{n_X + n_Y - 2}$$

תחת H0 נציב μY − μX = 0.

רווח סמך להפרש התוחלות:

$$\left[ (\bar Y - \bar X) \pm t_{n_X+n_Y-2,1-\alpha/2} \cdot \sqrt{S_p^2 \left(\frac{1}{n_X} + \frac{1}{n_Y}\right)}\right]$$

5 · מבחן F לשוויון שונויות

📘המבחן המקדים

H0: σX2 = σY2 נגד H1: σX2 ≠ σY2.

סטטיסטי המבחן:

$$F = \frac{S_X^2}{S_Y^2} \sim F(n_X - 1,\ n_Y - 1)$$

תכונות התפלגות F:

  • ערכים ב־[0, ∞).
  • שני פרמטרים: דרגות חופש במונה ובמכנה.
  • לא סימטרית.
  • אין נוסחאות סגורות — טבלאות או scipy.

אם F רחוק מ־1 (לכל אחד מהכיוונים) → דוחים את שוויון השונויות.

📋סדר העבודה בשני מדגמים בלתי־תלויים
  1. חישוב X̄, Ȳ, SX2, SY2.
  2. מבחן F לשוויון שונויות.
  3. אם לא דחינו שוויון → pooled t. אם דחינו → Welch t.
  4. השוואה ל־α הראשי של מבחן ההפרש בתוחלות.

6 · תרגילי מבחן

📝תרגיל 1 — מדגם בודד (תרגיל בית 8, שאלה 1)

מרחק קפיצה בחלל: N(20, 4) שנות אור. ספינה הגיעה ליקום מקבילי וטוענים שתוחלת המרחק קטנה. ב־5 קפיצות נמדדו: 18, 17, 18, 19, 20. בדקו ברמת מובהקות 5%.

H0: μ = 20, H1: μ < 20. n = 5, α = 0.05.

הכנה — חישובי מדגם:

∑Xi = 92, X̄ = 18.4

∑(Xi − X̄)2 = (−0.4)2+(−1.4)2+(−0.4)2+(0.6)2+(1.6)2 = 5.20

S2 = 5.20/4 = 1.30, S = 1.140

הערה: בשאלה נאמר "אולי גם סטיית התקן השתנתה" → אי אפשר להניח שהשונות עדיין 4 → לא ידועה → מבחן t (לא z).

שיטה 1 — סטטיסטי t מול ערך קריטי

ts = (X̄ − μ0) / (S/√n) = (18.4 − 20) / (1.140/√5) = −1.6 / 0.5099 ≈ −3.14

ערך קריטי (חד־צדדי שמאלי, df=4, α=0.05): −t4, 0.95 = −2.132. −3.14 < −2.132 → דוחים.

שיטה 2 — אזור דחייה C על X̄

C = {X̄ ≤ μ0 − t4, 0.95 · S/√n} = {X̄ ≤ 20 − 2.132·0.5099} = {X̄ ≤ 18.913}

הממוצע X̄ = 18.4 < 18.913 → בתוך אזור הדחייה → דוחים.

שיטה 3 — p-value

p.v = P(t4 ≤ −3.14) ≈ 0.0173 (מ־scipy.stats.t.cdf). 0.0173 < 0.05 → דוחים.

שלוש השיטות מסכימות: תוחלת המרחק קטנה ביקום המקבילי.

📝תרגיל 2 — בדיקה דו־צדדית + רווח סמך (דוגמה 2 מההרצאה)

ציוני 9 תלמידים: 67, 60, 71, 76, 63, 70, 85, 60, 82. בשנים קודמות הממוצע 79. (א) בדקו ירידה ברמת 5%. (ב) בדקו שינוי כללי ברמת 5%. (ג) רווח סמך 95% לתוחלת.

הכנה — חישובי מדגם: ∑Xi = 634, X̄ = 70.44, S2 = 82.78, S = 9.10. S/√n = 9.10/3 ≈ 3.03.

(א) חד־צדדי שמאלי. H0: μ = 79, H1: μ < 79.

שיטה 1 — סטטיסטי t מול ערך קריטי:

ts = (70.44 − 79) / 3.03 ≈ −2.82. קריטי −t8, 0.95 = −1.860. −2.82 < −1.860 → דוחים.

שיטה 2 — אזור דחייה C:

C = {X̄ ≤ 79 − 1.860·3.03} = {X̄ ≤ 73.36}. X̄ = 70.44 < 73.36 → דוחים.

שיטה 3 — p-value:

p.v = P(t8 ≤ −2.82) ≈ 0.0112. 0.0112 < 0.05 → דוחים.

(ב) דו־צדדי. H0: μ = 79, H1: μ ≠ 79.

שיטה 1: אותו |ts| = 2.82. קריטי t8, 0.975 = 2.306. 2.82 > 2.306 → דוחים.

שיטה 2 — אזור דחייה C:

C = {X̄ ≤ 79 − 2.306·3.03} ∪ {X̄ ≥ 79 + 2.306·3.03} = {X̄ ≤ 72.01} ∪ {X̄ ≥ 85.99}

X̄ = 70.44 < 72.01 → באזור הדחייה השמאלי → דוחים.

שיטה 3 — p-value: p.v = 2 · 0.0112 ≈ 0.0225 < 0.05 → דוחים.

שיטה 4 — דרך רווח הסמך (תוצאת ג'): 79 ∉ [63.45, 77.43] → דוחים.

(ג) רווח סמך 95%:

[X̄ ± t8, 0.975 · S/√n] = [70.44 ± 2.306·3.03] = [70.44 ± 6.99] = [63.45, 77.43]

79 מחוץ → מאשר את הדחייה בסעיף (ב).

📝תרגיל 3 — מדגם מזווג (דוגמה 3 מההרצאה)

בן מדדה 4 שבועות פיצוח גרעינים וצפייה בטלוויזיה:

שבועגרעינים (X)טלוויזיה (Y)D = Y − X

23.13.0−0.1
32.73.00.3

בדקו ברמת 0.1 שטלוויזיה ארוכה יותר.

זה מדגם מזווג (אותו ילד בכל שבוע) → t על D. n = 4, α = 0.1.

H0: μD = 0, H1: μD > 0 (טלוויזיה ארוכה יותר).

הכנה — חישובים על ההפרשים:

D̄ = (0.4 − 0.1 + 0.3 + 0.3)/4 = 0.9/4 = 0.225

∑(D − D̄)2 = (0.175)2+(−0.325)2+(0.075)2+(0.075)2 = 0.1475

SD2 = 0.1475/3 = 0.0492, SD = 0.222. SD/√n = 0.222/2 = 0.111.

שיטה 1 — סטטיסטי t מול ערך קריטי

ts = (D̄ − 0) / (SD/√n) = 0.225 / 0.111 = 2.03

קריטי (חד־צדדי ימני, df=3): t3, 0.9 = 1.638. 2.03 > 1.638 → דוחים.

שיטה 2 — אזור דחייה C על D̄

C = {D̄ ≥ 0 + t3, 0.9·SD/√n} = {D̄ ≥ 1.638·0.111} = {D̄ ≥ 0.182}

D̄ = 0.225 ≥ 0.182 → באזור הדחייה → דוחים.

שיטה 3 — p-value

p.v = P(t3 ≥ 2.03) ≈ 0.0676. 0.0676 < 0.1 → דוחים.

(שימו לב: ברמת 5% לא היו דוחים כאן — 0.0676 > 0.05.) תוחלת זמן הטלוויזיה ארוכה יותר ברמת 10%.

📝תרגיל 4 — שני מדגמים בלתי־תלויים + מבחן F (תרגיל בית 9, שאלה 3)

נמל אשדוד: n=21, X̄=5500, SX=600. נמל חיפה: n=11, Ȳ=4900, SY=900. בדקו ברמת 0.01 אם תוחלות שוות.

שלב 1: מבחן F לשוויון שונויות.

Fs = SY2/SX2 = 9002/6002 = 810,000/360,000 = 2.25. (לוקחים את הגדול במונה כדי לחפש בזנב הימני.)

קריטי דו־צדדי F10, 20, 0.995 ≈ 3.85 (טבלת F). 2.25 < 3.85 → לא דוחים שוויון שונויות.

שלב 2: pooled t-test.

Sp2 = [(20)·360,000 + (10)·810,000] / 30 = [7,200,000 + 8,100,000]/30 = 510,000. Sp ≈ 714.1.

ts = (5500 − 4900) / √(510,000·(1/21 + 1/11)) = 600 / √(510,000·0.1385) = 600 / 266.0 = 2.26

קריטי דו־צדדי t30, 0.995 ≈ 2.750. 2.26 < 2.750 → לא דוחים H0 ברמת 1%. אין מספיק ראיות להבדל בהכנסות.

(הערה: ברמת 5% — קריטי t30, 0.975 ≈ 2.042, ושם היו דוחים.)
